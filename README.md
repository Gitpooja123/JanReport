JanReport is a full-stack civic issue reporting and monitoring system where citizens can report public problems and government authorities (MLA/MP) can track, analyze, and prioritize them for faster resolution.

---

##  Modules

- **Citizen Module**  
  Users can register, login, report civic issues, track their status, and communicate with authorities.

- **Admin Module**  
  Admin verifies issues, manages users, and monitors overall system activity.

- **MLA/MP Dashboard**  
  Authorities can view constituency-wise reports, issue trends, map-based hotspots, and priority analytics.
